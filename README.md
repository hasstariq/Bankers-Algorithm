## Steps to run

Make sure the Makefile, Source File and Sample File are in the same directory. 
Use the following commands:<br />
"mv Makefile.mk makefile"<br />
"make warnings_OK"<br />
"./main 10 10 10 10"<br />
(Any value for resources can be used instead of 10)
